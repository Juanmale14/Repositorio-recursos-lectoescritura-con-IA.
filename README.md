Recursos Digitales para la Lectoescritura

Este proyecto es un repositorio interactivo y moderno diseñado para centralizar recursos, herramientas de IA y documentos metodológicos enfocados en el trabajo de la lectoescritura en el aula.

La aplicación se presenta como una Single Page Application (SPA) de archivo único, construida con estándares web modernos sin dependencias externas, lo que garantiza una carga ultrarrápida y una portabilidad total.

🚀 Características

Diseño Premium Moderno: Interfaz limpia con estética Glassmorphism, sombras suaves y tipografía de sistema optimizada.

Buscador Inteligente: Filtro en tiempo real que busca por título, descripción o contenido de etiquetas.

Nube de Etiquetas Interactiva: Sistema de filtrado rápido mediante tags para categorizar los recursos.

Totalmente Responsivo: Adaptado para una visualización perfecta en dispositivos móviles, tablets y ordenadores de escritorio.

Arquitectura Zero-Dependency: Escrito en HTML5, CSS3 y JavaScript Vanilla. No requiere Node.js, librerías externas ni frameworks.

🛠️ Tecnologías utilizadas

HTML5: Estructura semántica para accesibilidad y SEO.

CSS3: Variables CSS (Custom Properties), Flexbox y CSS Grid para el layout responsivo.

JavaScript (ES6+): Lógica de filtrado dinámico y manipulación del DOM.

📦 Instalación y Uso

Al ser un archivo único, no requiere instalación:

Descarga el archivo index.html.

Ábrelo en cualquier navegador moderno (Chrome, Firefox, Edge, Safari).

Si deseas subirlo a GitHub Pages:

Sube el archivo index.html a un repositorio de GitHub.

Ve a Settings > Pages.

Selecciona la rama main y guarda. ¡Tu repositorio estará online en segundos!

📝 Estructura de Datos

Los recursos están definidos en una constante data dentro del script principal. Cada objeto sigue este esquema:

{
    titulo: "Nombre del Recurso",
    descripcion: "Breve explicación de su utilidad didáctica.",
    enlace: "[https://url-del-recurso.com](https://url-del-recurso.com)",
    etiquetas: ["Categoría1", "Categoría2"],
    imagen: "URL de la imagen de portada"
}
